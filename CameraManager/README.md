## Camera Manager

메인 카메라를 제어하기 위한 싱글톤 기반 시스템입니다.  
카메라의 위치, 회전, 연출 및 다양한 렌더링 옵션을 통합적으로 관리합니다.

---

### 구조

카메라는 다음과 같은 계층 구조로 구성됩니다.

- Root Object (Focus 추적)
  - Camera Object (실제 렌더링 카메라)

Root Object가 촬영 대상(유닛)을 따라가고,  
카메라는 해당 오브젝트의 자식으로 동작합니다.

---

### 설계 의도

카메라의 이동과 회전을 직접 제어하는 대신,  
Root Object를 기준으로 카메라를 제어하도록 구조를 분리했습니다.

이를 통해:

- 카메라 거리 조절 (Zoom)
- 회전 처리
- 위치 보정

등을 보다 직관적으로 처리할 수 있습니다.

---

### 주요 기능

- 카메라 위치 및 회전 제어
- 줌 인 / 줌 아웃
- 카메라 흔들림(Shake) 연출
- 스카이박스 설정
- 레이어 컬링 제어
- 깊이 컬링 설정
- 오디오 리스너 제어

Unity에서 제공하는 카메라 관련 기능을 통합적으로 제어할 수 있도록 구성했습니다.

---

### 카메라 모드 시스템

카메라의 동작 방식을 상황에 따라 변경할 수 있도록 설계했습니다.

지원 모드 예시:

- 즉시 추적 (Snap)
- 부드러운 추적 (Smooth Follow)
- 1인칭 시점 (First Person)

---

### 확장성

- 추적 방식 변경을 통해 다양한 카메라 연출 대응 가능
- 새로운 카메라 모드 추가가 용이한 구조
- 다양한 게임 상황(전투, 탐험 등)에 맞는 카메라 제어 가능

---

### 확인 포인트

- Root 기반 카메라 제어 구조
- 추적 방식(즉시 / 보간) 분리
- 카메라 기능을 단일 매니저에서 통합 관리한 설계
