## UI System

게임 UI를 구성하고 관리하기 위한 시스템입니다.  
버튼, 패널, 조이스틱 등 다양한 UI 요소를 공통 구조로 관리하며,  
입력 및 이벤트 시스템과 연동되도록 설계되었습니다.

---

### 개요

모든 UI는 공통 베이스 클래스를 기반으로 동작하며,  
입력 이벤트 및 게임 이벤트를 처리하고 전파할 수 있습니다.

또한 UI는 계층 구조를 통해 구성되며,  
상위 UI가 하위 UI를 관리하는 방식으로 동작합니다.

---

### UI 구조

UI는 컨테이너 기반 구조로 설계되어 있습니다.

- UI Container (부모)
  - UI Element (자식)

---

### 컨테이너 시스템

각 UI는 자신을 컨테이너로 사용하여  
다른 UI 요소를 등록할 수 있습니다.

📌 예시

```csharp
AddElement<UIxButton>("ResumeBtn", ...);
AddElement<UIxButton>("StatusBtn", ...);
